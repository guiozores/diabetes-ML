Sobre o Projeto

O que é o DiabetesML?

O DiabetesML é uma aplicação web desenvolvida para auxiliar na avaliação preliminar do risco de diabetes. Utilizando técnicas de machine learning treinadas com o dataset Pima Indians Diabetes, este sistema oferece uma estimativa do risco de desenvolvimento da doença com base em diversos fatores clínicos.

Como funciona?

O sistema analisa dados como níveis de glicemia, histórico familiar, IMC, idade e outros fatores relevantes para calcular o risco de diabetes. O algoritmo de Machine Learning (Random Forest) foi treinado com dados reais de pacientes do dataset Pima Indians Diabetes, reconhecido pela comunidade científica.

O que este sistema não é

Atenção: Esta ferramenta não substitui diagnóstico médico profissional. Os resultados devem ser considerados apenas como uma ferramenta educativa e de conscientização. Se você tem preocupações sobre sua saúde, consulte um médico para avaliação adequada.

Nosso Objetivo

O objetivo principal deste projeto é aumentar a conscientização sobre os fatores de risco para diabetes e encorajar a procura por assistência médica quando necessário. A ferramenta pode ser especialmente útil para:

  • Conscientização sobre fatores de risco para diabetes
  • Educação em saúde sobre indicadores clínicos importantes
  • Incentivo à prevenção e detecção precoce da doença
  • Suporte à decisão preliminar sobre a necessidade de consulta médica
